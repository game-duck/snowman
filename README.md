## 💻 About The Project

친구들에게 눈사람을 만들어 편지와 함께 전달할 수 있습니다!</br>
로그인/회원가입으로 생성된 본인만의 정원을 링크로 친구들에게 공유할 수 있으며, </br>
친구들이 만들어준 개성있는 눈사람들을 내 정원에서 한 눈에 확인해 보세요!   

| 제작 의도 | 스노우맨가든은 친구들과 즐겁게 소통하기 위해 제작한 롤링페이퍼 서비스입니다.|
